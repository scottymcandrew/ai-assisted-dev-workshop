# AI Dev Workshop

A short, hands-on workshop on writing effective prompts for AI coding agents.

## Setup

```bash
./setup.sh
```

This configures Claude Code to use the workshop's shared Amazon Bedrock account so you don't need to set up your own AWS credentials.
